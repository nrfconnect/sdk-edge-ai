# Edge AI Add-on for nRF Connect SDK

A compact add-on that makes it easy to run small, generated machine-learning
models on Nordic Semiconductor devices using the nRF Connect SDK (NCS).

Releases
--------

The latest release of the Edge AI Add-on is **v2.0.1**.
This is a patch release on top of v2.0.0 that updates the Axon NPU software to v1.2.0 to restore compatibility with the Nordic Edge AI Lab web tool.

For the full list of changes, see the [release notes](https://docs.nordicsemi.com/bundle/addon-edge-ai_latest/page/doc/release_notes.html).
