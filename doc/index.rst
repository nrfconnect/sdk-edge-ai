Welcome to the Edge AI add-on for |NCS|
#######################################

This is add-on for `nRF Connect SDK`_.

The Edge AI add-on contains solutions and integrations for running machine learning models on Nordic devices.

.. note::
   Until existing support for machine learning models is removed for nrf there is a need to apply patch for nrf repository.
   Patch file is provided and can be applied by executing ``west patch apply``.
